## 🪄 还原
由于版权原因，本项目无法直接为您提供来自《我的世界》的资产 *（如图像、音频、字体等）*，
所以项目只能使用开源或免费可商用的内容**替代**原内容，因此项目对与此方面无法完全还原。

但项目提供了修改默认使用资产的方式，**推荐**使用以下方式对内容进行设置：
```kotlin
/*vip.cdms.orecompose.style.*/OreTheme(/*...*/) {
    // 在这里使用 OreCompose 组件...
}
```
